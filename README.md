# futbol
son dos proyectos. en "FUTBOL.R", aparece el código para mapear pases en un torneo. en este caso, utilicé los datos abiertos de statsbomb. los datos mostrados son de aitana bonmatí en la copa mundial del 2023. 
en "datos_html_tiros.R", hago una correlación sobre los toques en área rival y los goles esperados (xG) en la liga mexicana de futbol varonil. en el código mostrado, hago webscrapping de fbref.com y un par de gráficas utilizando ggplot.
